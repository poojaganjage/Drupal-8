ABOUT RMS Theme
------------

RMS Theme is the default theme for RMS.  It is a flexible,
with a responsive and mobile-first layout, supporting 8 regions.

This theme is not intended to be used as a base theme.

Setup for further development/modifications:
1. Download latest version of RMS distribution from
https://www.drupal.org/project/rms
2. Navigate to profiles/rms/themes/rms.
3. Run yarn install
4. Run gulp watch to reload the site with your changes!
